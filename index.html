忘れたときに見返す用。分散・SD・SE・信頼区間の連鎖と各検定との関係をまとめています。

① 統計量の連鎖 — Variance → SD → SE → CI
1
分散
Variance
各サンプルと平均の距離を二乗して平均したもの
そのまま足すと0になるので二乗する
Σ(xᵢ − x̄)² / n
例：[3,5,5,7,5] 平均=5
距離²=[4,0,0,4,0]
→ 分散 = 8/5 = 1.6
√
2
標準偏差
Standard Deviation (SD)
分散の√。単位を元に戻したもの
「1人1人が平均からどのくらい離れているか」
SD = √Variance
√1.6 = 1.26
→「1人1人が平均から約1.26点離れている」
÷√n
3
標準誤差
Standard Error (SE)
「別のサンプルを集めたとき平均がどのくらいブレるか」
SDが大きい（個人差大）→ブレやすい
nが大きい（サンプル多）→安定する
この2つの性質をSD÷√nにまとめている
SE = SD / √n
SD=1.26, n=5
→ SE = 1.26 ÷ √5 = 0.56
→「別の5人を集めても平均は±0.56点程度しかブレない」
×1.96
4
信頼区間
Confidence Interval (CI)
真の平均が95%の確率で収まる範囲
1.96は正規分布の性質（平均±1.96SDに95%が入る）
n≥30なら中心極限定理で平均も正規分布に従う
→ SDの代わりにSEを使えばOK
95%CI = 平均 ± 1.96 × SE
平均=5, SE=0.56
1.96×0.56 = 1.10
→ CI = [3.90, 6.10]
→「95%の確率で真の平均はこの範囲」
正規分布
仮定不可
5
ブートストラップCI
Bootstrap Confidence Interval
正規分布を仮定できない場合（Ridge回帰など）
同じデータから1000回復元抽出して統計量を計算
下位2.5%〜上位97.5%を信頼区間とする
「数式ではなく実際に繰り返してみて実測した範囲」
下位2.5% 〜 上位97.5%
（1000回リサンプリング）
例：別府ダミーの係数を1000回計算
→ 下位2.5%=0.050, 上位97.5%=0.342
→「0をまたがないので有意」
② 各手法とSEの関係
手法 English SEとの関係・仕組み
t検定t-test compare 平均の差 ÷ SE = t値 → t分布でp値に変換
例：幸福度の差1.37 ÷ 合成SE0.292 = t値4.69 → p≒0.0000（偶然この差が起きる確率がほぼゼロ）
nが大きいほどt分布は正規分布に近づく。t検定はANOVAの2グループ版で F = t² の関係がある
分散分析ANOVA compare F値 = グループ間分散 ÷ グループ内分散 → F分布でp値に変換
グループ間分散：各グループの平均が全体平均から離れている度合い（二乗平均）
グループ内分散：各グループ内の個人差（SD）の平均
F=1.0 → グループ分けに意味がない　F=8.53 → グループ間の差が個人差の8.53倍
多重比較Tukey HSD post-hoc ANOVAで「どこかに差がある」とわかった後に「どのペアに差があるか」を特定
なぜt検定を繰り返してはダメか：α=0.05は「1回で偽陽性5%以内」の約束。15回繰り返すと1-(0.95)¹⁵=54%で偶然有意になる
Tukey HSDは「15ペア全体として偽陽性5%以内」になるようp値の基準を厳しくする → p-adjはt検定のp値より大きくなる
重回帰分析OLS Regression regression 係数 ÷ SE = t値 → t分布でp値に変換
係数・p値・R²で結果を解釈する
Ridge回帰Ridge Regression regularize 正則化でSEが計算不可 → ブートストラップCIで不確実性を推定
多重共線性がある場合に使う正則化回帰
③ t検定 vs Mann-Whitney U検定
t検定
比較対象平均値
前提正規分布が必要
弱点外れ値に引っ張られる
Mann-Whitney U検定
比較対象順位の偏り
前提正規分布不要
強み外れ値の影響を受けない
なぜ正規分布が不要なのか（ノンパラメトリックの仕組み）
全データをひとまとめにして小さい順にランキングする。100という外れ値も「1番大きい＝8位」になるだけで、それ以上の影響を持たない。
順位は「大小関係」しか持たないため、分布の形に左右されない → だからノンパラメトリック検定と呼ばれる。

U値：全国×別府の全ペア総当たりで「別府が上位だった回数」を数えたもの。最大値 = 188×113 = 21,244
p値：今回のU値が「差がないとき」に偶然生じる確率。U値が端に偏るほどp値は小さくなる
効果量 r：U値だけでは差の大きさが不明 → r = |Z| / √N で補う（0.1未満:なし／0.1〜0.3:小／0.3〜0.5:中／0.5以上:大）

確認したこと結果判断
Shapiro-Wilk検定両グループとも p < 0.05非正規分布と判定
W統計量別府0.942、全国0.9561に近く極端な歪みではない
