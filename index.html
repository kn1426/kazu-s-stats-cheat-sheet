忘れたときに見返す用。分散・SD・SE・信頼区間の連鎖と各検定との関係をまとめています。

A quick-reference sheet for when you forget. Covers the Variance → SD → SE → CI chain and how each test relates to SE.

① 統計量の連鎖 — Variance → SD → SE → CI① Statistic Chain — Variance → SD → SE → CI
1
分散
Variance
各サンプルと平均の距離を二乗して平均したもの
そのまま足すと0になるので二乗する
Average of squared distances between each sample and the mean.
Squaring prevents them from summing to zero.
Σ(xᵢ − x̄)² / n
例：[3,5,5,7,5] 平均=5
距離²=[4,0,0,4,0]
→ 分散 = 8/5 = 1.6
e.g. [3,5,5,7,5] mean=5
dist²=[4,0,0,4,0]
→ Variance = 8/5 = 1.6
√
2
標準偏差
Standard Deviation (SD)
分散の√。単位を元に戻したもの
「1人1人が平均からどのくらい離れているか」
Square root of variance — restores the original units.
"How far is each individual from the mean?"
SD = √Variance
√1.6 = 1.26
→「1人1人が平均から約1.26点離れている」
√1.6 = 1.26
→ "Each person deviates ~1.26 pts from the mean"
÷√n
3
標準誤差
Standard Error (SE)
「別のサンプルを集めたとき平均がどのくらいブレるか」
SDが大きい（個人差大）→ブレやすい
nが大きい（サンプル多）→安定する
この2つの性質をSD÷√nにまとめている
"How much would the mean fluctuate if we drew a new sample?"
Large SD (high variability) → more fluctuation
Large n (more data) → more stable
Both properties captured in SD ÷ √n
SE = SD / √n
SD=1.26, n=5
→ SE = 1.26 ÷ √5 = 0.56
→「別の5人を集めても平均は±0.56点程度しかブレない」
SD=1.26, n=5
→ SE = 1.26 ÷ √5 = 0.56
→ "Even a fresh sample of 5 gives a mean within ±0.56"
×1.96
4
信頼区間
Confidence Interval (CI)
真の平均が95%の確率で収まる範囲
1.96は正規分布の性質（平均±1.96SDに95%が入る）
n≥30なら中心極限定理で平均も正規分布に従う
→ SDの代わりにSEを使えばOK
Range within which the true mean falls 95% of the time.
1.96 comes from the normal distribution (mean ± 1.96 SD covers 95%).
For n ≥ 30, the CLT ensures the sample mean is normal.
→ Substitute SE for SD.
95%CI = 平均mean ± 1.96 × SE
平均=5, SE=0.56
1.96×0.56 = 1.10
→ CI = [3.90, 6.10]
→「95%の確率で真の平均はこの範囲」
mean=5, SE=0.56
1.96×0.56 = 1.10
→ CI = [3.90, 6.10]
→ "95% chance the true mean is in this range"
正規分布
仮定不可normality
not met
5
ブートストラップCI
Bootstrap Confidence Interval
正規分布を仮定できない場合（Ridge回帰など）
同じデータから1000回復元抽出して統計量を計算
下位2.5%〜上位97.5%を信頼区間とする
「数式ではなく実際に繰り返してみて実測した範囲」
Used when normality cannot be assumed (e.g. Ridge regression).
Resample from the same data 1000 times, compute the statistic each time.
Use the 2.5th–97.5th percentile as the CI.
"Empirically measured by repetition — no formula needed."
下位lower 2.5% 〜 上位upper 97.5%
(1000回リサンプリング1000 resamples)
例：別府ダミーの係数を1000回計算
→ 下位2.5%=0.050, 上位97.5%=0.342
→「0をまたがないので有意」
e.g. Beppu dummy coefficient over 1000 resamples
→ 2.5th pct=0.050, 97.5th pct=0.342
→ "Does not straddle 0 → significant"
② 各手法とSEの関係② How Each Method Relates to SE
手法Method SEとの関係・仕組みRelationship to SE / Mechanism
t検定t-testt-test 平均の差 ÷ SE = t値 → t分布でp値に変換
例：幸福度の差1.37 ÷ 合成SE0.292 = t値4.69 → p≒0.0000（偶然この差が起きる確率がほぼゼロ）
nが大きいほどt分布は正規分布に近づく。t検定はANOVAの2グループ版で F = t² の関係がある Difference in means ÷ SE = t-statistic → converted to p via t-distribution
e.g. happiness diff 1.37 ÷ pooled SE 0.292 = t=4.69 → p≈0.0000 (near-zero chance this occurs randomly)
As n grows, t-distribution approaches normal. t-test is ANOVA for 2 groups: F = t²
分散分析ANOVAANOVA F値 = グループ間分散 ÷ グループ内分散 → F分布でp値に変換
グループ間分散：各グループの平均が全体平均から離れている度合い（二乗平均）
グループ内分散：各グループ内の個人差（SD）の平均
F=1.0 → グループ分けに意味がない　F=8.53 → グループ間の差が個人差の8.53倍 F = between-group variance ÷ within-group variance → p via F-distribution
Between-group variance: how far each group mean strays from the grand mean (squared)
Within-group variance: average individual variability (SD) across groups
F=1.0 → grouping adds nothing; F=8.53 → between-group difference is 8.53× individual variance
多重比較Tukey HSDTukey HSD ANOVAで「どこかに差がある」とわかった後に「どのペアに差があるか」を特定
なぜt検定を繰り返してはダメか：α=0.05は「1回で偽陽性5%以内」の約束。15回繰り返すと1-(0.95)¹⁵=54%で偶然有意になる
Tukey HSDは「15ペア全体として偽陽性5%以内」になるようp値の基準を厳しくする → p-adjはt検定のp値より大きくなる After ANOVA finds "some difference exists," identifies which specific pairs differ.
Why not repeat t-tests? α=0.05 guarantees ≤5% false positive per test. With 15 tests: 1−(0.95)¹⁵=54% chance of a spurious result.
Tukey HSD tightens the p-value threshold so the family-wise false-positive rate stays ≤5% → p-adj is larger than raw t-test p-values.
重回帰分析OLS RegressionOLS Regression 係数 ÷ SE = t値 → t分布でp値に変換
係数・p値・R²で結果を解釈する Coefficient ÷ SE = t-statistic → converted to p via t-distribution
Interpret results via coefficients, p-values, and R²
Ridge回帰Ridge RegressionRidge Regression 正則化でSEが計算不可 → ブートストラップCIで不確実性を推定

Regularized regression for handling multicollinearity

なぜ正規分布が不要なのか（ノンパラメトリックの仕組み）Why No Normality Needed (How Non-Parametric Tests Work)
全データをひとまとめにして小さい順にランキングする。100という外れ値も「1番大きい＝8位」になるだけで、それ以上の影響を持たない。
順位は「大小関係」しか持たないため、分布の形に左右されない → だからノンパラメトリック検定と呼ばれる。

U値：全国×別府の全ペア総当たりで「別府が上位だった回数」を数えたもの。最大値 = 188×113 = 21,244
p値：今回のU値が「差がないとき」に偶然生じる確率。U値が端に偏るほどp値は小さくなる
効果量 r：U値だけでは差の大きさが不明 → r = |Z| / √N で補う（0.1未満:なし／0.1〜0.3:小／0.3〜0.5:中／0.5以上:大） All data are pooled and ranked in ascending order. An outlier of 100 simply becomes "rank 8 (largest)" — no extra leverage beyond its rank.
Ranks encode only ordering, so the result is independent of distribution shape → hence called a non-parametric test.

U statistic: counts the number of times a Beppu value outranks a national value across all pairs. Maximum = 188×113 = 21,244.
p-value: probability of observing this U value by chance when there is no true difference. The more extreme the U, the smaller the p.
Effect size r: U alone does not tell you the magnitude → supplement with r = |Z| / √N (<0.1: none / 0.1–0.3: small / 0.3–0.5: medium / ≥0.5: large)
